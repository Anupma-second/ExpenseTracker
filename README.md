# 💰 ExpenseTracker

An Android expense tracking app built with:
- Kotlin
- Jetpack Compose
- Firebase Auth + Firestore

## Features
- Login / Register
- Add, edit, delete income and expense transactions
- Category management
- Monthly budgets with progress tracking
- Reports with pie chart
- Real-time sync with Firebase

## Setup Instructions

1. Clone this repository
2. Open in Android Studio
3. Create a Firebase project at https://console.firebase.google.com
4. Add an Android app with package name `com.yourname.expensetracker`
5. Download `google-services.json` and place it in the `app/` folder
6. Enable Firebase Authentication (Email/Password)
7. Enable Firestore Database
8. Run the app

## Requirements
- Android Studio Hedgehog or newer
- Minimum SDK: API 26 (Android 8.0)
- Google account for Firebase
